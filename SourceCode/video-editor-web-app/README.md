# Web Setup

This web app intentionally uses no framework and has a single entry point.

## Required files

- `index.html`
- `style.css`
- `app.js`

## Dependencies

- No runtime package dependencies.
- Optional local server for development (example: `npx serve`).

## Install

No install required for static run.

## Run

Option 1 (double click):
- Open `index.html` directly in a browser.

Option 2 (recommended local server):
- Run: `npx serve SourceCode/video-editor-web-app -l 4173`
- Open: `http://localhost:4173`
